Module 1 · 45 min Context & Mindset
Why Spatial Thinking Changes Everything
+
💡 Most GIS courses open the software on minute one. This one starts with the question that matters: why does location data give you an insight advantage — and what does that mean for your career or business?
1
The Spatial Advantage
Why the majority of business data has a location component that most people ignore. Show real examples: flood risk planning, retail site selection, disease outbreak mapping, environmental monitoring. Location context turns raw data into decisions.
2
Installing & Configuring QGIS
Download from qgis.org (free, open-source, cross-platform). Long-term release vs latest version — which to choose. First-time settings: dark theme, autosave interval, projection defaults. Install the two essential starter plugins: QuickMapServices and QuickOSM.
3
The QGIS Interface — Every Panel Explained
Layers panel, Map Canvas, Browser panel, Status bar, Toolbars. What each does and when you use it. How to dock, undock, and reset panels. Saving your workspace as a .QGZ project file. Understanding what a project file does — and doesn't — save.
4
Coordinate Reference Systems — Demystified
The crumpled-orange-peel explanation of why all flat maps distort reality. Geographic CRS (WGS84 / EPSG:4326) vs Projected CRS. How to check the CRS of a layer. How to set the project CRS. On-the-fly reprojection. Why mismatched CRS is the #1 beginner mistake.
◆Explain why spatial thinking creates competitive insight across industries
◆Install and correctly configure QGIS from scratch
◆Navigate all major QGIS interface panels with confidence
◆Understand and set Coordinate Reference Systems correctly
◆Load your first data layer and save a working project file
Hands-On Exercise
Add a basemap from OpenStreetMap using QuickMapServices. Create a CSV spreadsheet of 10 real locations meaningful to you (cafes, offices, parks). Load it into QGIS using Add Delimited Text Layer. Change each point's symbology to a coloured circle. Save your project. Navigate: zoom, pan, identify features.
Free Resources
qgis.org (download) · QuickMapServices plugin (basemaps) · Any spreadsheet app to create your CSV

Module 2 · 60 min Spatial Data Literacy
Data: Finding It, Reading It, Trusting It
+
💡 Bad data in = bad maps out. This module turns students into ruthless spatial data critics — able to source, assess, and join data from anywhere in the world.
1
Vector vs Raster — The Core Mental Model
Points, lines and polygons (vector) vs pixel grids (raster). When each data type is appropriate and why. Key file formats: Shapefile (.shp), GeoPackage (.gpkg), GeoJSON (.geojson), CSV with coordinates, GeoTIFF (.tif). GeoPackage as the modern preferred format and why it replaces Shapefiles.
2
The Free Global Data Ecosystem
Live walkthrough of six authoritative free sources: GADM for administrative boundaries, Natural Earth for global datasets, WorldPop for population grids, OpenStreetMap via QuickOSM plugin for streets and POIs, Copernicus Open Access Hub for satellite imagery, USGS EarthExplorer for elevation (SRTM). Build a reusable personal data library on your hard drive.
3
Attribute Table Mastery
Opening, sorting, filtering the attribute table. The Field Calculator: create new columns, calculate area, length, and perimeter in real-world units. Write basic expressions. Select features by attribute. Fix messy field names. Understand data types: string, integer, double, date.
4
Table Joins — Connecting the Dots
Join a non-spatial spreadsheet (CSV) to a polygon layer using a common field. One-to-one vs one-to-many joins. Common failure points: mismatched field types, extra spaces in values, encoding issues. How to make the join permanent by exporting. A joined dataset is the foundation of almost every choropleth map.
◆Distinguish and work with vector and raster data types and their file formats
◆Source free authoritative spatial data for any country or region
◆Inspect, filter, and edit attribute tables with confidence
◆Calculate new fields using the QGIS Field Calculator
◆Join external tabular data to a spatial layer and diagnose common errors
Hands-On Exercise
Download country boundaries from GADM and a CSV of World Bank GDP per capita statistics. Join them using the country code field. Calculate a new column showing GDP per capita rounded to the nearest thousand. Classify countries into four income groups and export the result as a GeoPackage. Inspect which join rows failed and why.
Free Data Sources
gadm.org · data.worldbank.org · naturalearthdata.com · QuickOSM plugin (built into QGIS)

Module 3 · 75 min Visualisation & Symbology
Cartography That Communicates
+
💡 Anyone can put data on a map. The skill is designing a map that communicates clearly, quickly, and honestly. This module is a practical design education built on cartographic principles used by national mapping agencies.
1
Five Cartographic Principles Every Map Needs
Visual hierarchy (what the eye reads first), figure-ground contrast, colour theory for data (sequential, diverging, qualitative schemes), text placement rules, and the discipline of knowing what to leave out. Analyse three real published maps — what works, what misleads, and why.
2
QGIS Symbology — Every Mode Explained
Single symbol, Categorised, Graduated, Rule-based, Heatmap, and 2.5D renderer. When each is appropriate. Colour ramp selection using ColorBrewer principles. Accessible palette choices for colour-blind readers. Transparency, blending modes, and layer effects for polished output.
3
Labels That Work
Label expressions using the QGIS expression engine. Placement modes: around point, on line, inside polygon. Priority and obstacle settings. Label callouts and connectors. When to use labels vs tooltips vs a legend. The PAL labeling engine's automatic conflict resolution.
4
Print Layout — Your Professional Deliverable
The QGIS Print Layout Manager. Adding and configuring map frames. Legends: what to include, what to exclude, how to reorder and rename items. Scale bars, north arrows, graticules. Adding your logo, title, data source credit, and projection note. Export settings: PNG at 300dpi for print, SVG for editing, PDF for sharing.
◆Apply the five core cartographic principles to any map
◆Configure all QGIS symbology modes and choose the right one for each dataset
◆Select accessible, perceptually correct colour palettes
◆Create map labels using expressions and control their placement
◆Design and export a print-ready A4 map layout at publication quality
Hands-On Exercise
Using the GDP dataset from Module 2, create a publication-quality choropleth map. Apply a sequential colour ramp with five graduated classes. Add country labels for the top 10 economies. Open Print Layout, add a title, legend, scale bar, north arrow, and data source credit. Export as a 300dpi PNG. Peer review: swap maps and critique using the five cartographic principles.
Free Resources
colorbrewer2.org (colour palettes) · fonts.google.com (free typefaces for layout) · Dataset from Module 2

Module 4 · 75 min Geoprocessing & Analysis
Spatial Analysis: Answering the "Where?" Question
+
💡 This is the module where QGIS stops being a map-making tool and becomes a decision-making tool. Every operation here maps directly to a real question an organisation needs answered.
1
The Core Geoprocessing Operations
Buffer: how far is X from Y? Clip: cut a dataset to a study area. Intersect: what features overlap? Union: merge two datasets. Dissolve: merge features by attribute. Difference: what's left after removing an area? Each explained with a practical scenario. Run them all via Vector menu and via the Processing Toolbox.
2
Spatial Queries and Selections
Select by Location: find features that intersect, are within, touch, or are nearest to another layer. Distance Matrix: calculate distances between point pairs. Count Points in Polygons: how many incidents per district? Join Attributes by Nearest: tag each point with data from the closest polygon. These four operations answer 80% of common GIS questions.
3
Raster Analysis & the Raster Calculator
Working with elevation data: hillshade, slope, aspect. The Raster Calculator for arithmetic and conditional operations on raster grids. Reclassify by table to convert continuous data into discrete categories. Extract raster values to points. Understand spatial resolution and its implications for analysis accuracy.
4
The QGIS Graphical Modeler
Build a reusable, documented, repeatable analysis workflow visually — no coding required. Chain multiple geoprocessing steps. Add input parameters so others can run the model with their own data. Export as a standalone QGIS model file. Introduction to why this matters for reproducible analysis and client reporting.
◆Execute the six essential geoprocessing operations and know when to use each
◆Run spatial queries to select, count, and tag features by location
◆Perform basic raster analysis including hillshade, slope and raster calculator
◆Build a reusable multi-step analysis model in the QGIS Graphical Modeler
◆Translate a real-world location question into a documented QGIS workflow
Hands-On Exercise
Site suitability analysis — find the best locations for a new community health clinic. Criteria: (1) more than 2km from existing clinics (buffer + difference), (2) high population density areas (raster classify), (3) within 500m of a main road (buffer + intersect), (4) within high-deprivation census areas (select by attribute + clip). Combine all criteria layers. Identify the top 3 candidate zones. Build the workflow in the Graphical Modeler so it can be rerun for any city.
Free Data Sources
OpenStreetMap roads & POIs via QuickOSM plugin · WorldPop population raster · Your country's census open data portal · GADM for admin boundaries

Module 5 · 45 min Output, Portfolio & Next Steps
Share Your Work & Keep Growing
+
💡 A map that only lives on your hard drive earns you nothing. This module publishes your work to the web, frames it as a portfolio piece, and maps out the exact skills that come next.
1
Publishing Interactive Maps for Free
Three zero-cost options: (1) qgis2web plugin — generates a standalone Leaflet or OpenLayers HTML file you host anywhere; (2) QGIS Cloud free tier — publish directly from QGIS; (3) Felt.com free tier — drag-and-drop GeoJSON upload with shareable link. When to use each. How to embed an interactive map iframe into a website or blog post.
2
Live Data Connections in QGIS
Connecting to WMS (Web Map Service) and WFS (Web Feature Service) live data feeds — data that updates automatically. Adding national mapping agency WMS feeds. OGC services: what they are and where to find them. QGIS as a client to PostGIS spatial databases. Introduction to SpatiaLite for local database storage.
3
Building a GIS Portfolio That Gets Noticed
The three project types that demonstrate real capability: a choropleth story map, a site suitability analysis, and a live data dashboard. How to write a project description (problem → method → result → insight). GitHub for version control and hosting HTML maps on GitHub Pages. LinkedIn positioning: how to describe QGIS skills clearly without jargon.
4
The QGIS Ecosystem & Your Learning Path
Essential plugins beyond the basics: MMQGIS (geocoding, joins), Semi-Automatic Classification Plugin (satellite image analysis), OpenLayers, Profile Tool. Community resources: GIS Stack Exchange, r/gis, QGIS mailing list, local QGIS user groups. Certificates worth pursuing: QGIS certifications, open university spatial courses. The honest next-skills roadmap: Python for QGIS (PyQGIS), PostGIS, remote sensing.
◆Publish an interactive web map using at least two free methods
◆Connect QGIS to live WMS and WFS data services
◆Describe and present your GIS work clearly to a non-technical audience
◆Identify five essential QGIS community resources to support continued learning
◆Know the exact skills that build on this course and how to access them
Hands-On Exercise
Export your Module 3 choropleth map as an interactive web map using qgis2web. Upload it to GitHub Pages or link via Felt. Write a 100-word portfolio project description: one sentence on the problem, two on your method, one on the result, one on the insight it reveals. Share the live link with the group. Final reflection: write down the one real-world spatial question in your own work or sector that you now know how to start answering.
Free Publishing Tools
